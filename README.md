This project is a Bash script created to automate the creation of:

- Users
- Groups
- Directories
- Permissions

## Features

✔ Creates organizational directories  
✔ Creates security groups  
✔ Assigns correct permissions  
✔ Creates users with encrypted passwords  
✔ Forces password change on first login  

## Technologies

- Bash Script
- Linux User Management
- OpenSSL
- chmod
- chown
- useradd
- groupadd
